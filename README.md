# EUAGELIA

Робочий проєкт для історико-критичної реконструкції Євангелія від Фоми логія-за-логією.

## Мета

Побудувати прозору реконструкцію шарів Євангелія від Фоми: не датувати всю книгу однією датою, а оцінювати кожну логію окремо за джерелами, паралелями, мовними ознаками, історичним контекстом і ймовірністю раннього походження.

## Структура

- `00_Project` - службові нотатки, плани, дорожня карта.
- `01_Methodology` - принципи реконструкції та критерії оцінки.
- `02_Sources_And_Manuscripts` - рукописи, мови, видання, онлайн-ресурси.
- `03_Scholarship_And_Dating` - історія досліджень, датування, позиції дослідників.
- `04_Historical_Hypotheses` - робочі історичні сценарії про Єшуа, ранні громади й Церкву.
- `05_Logia_Corpus` - майбутній корпус із 114 карток логій і зведеними таблицями.
- `06_Reconstructions` - зібрані реконструкції: Kernel Thomas, власна модель, варіанти шарів.
- `07_Bibliography` - бібліографія, цитати, джерела для перевірки.
- `08_Output` - майбутні тексти: книга, статті, публічні версії.
- `09_Archive` - матеріали, які більше не є активними, але мають бути збережені.

## Найближчий робочий цикл

1. Заповнити `05_Logia_Corpus/cards` картками для логій 1-114.
2. Для кожної логії внести коптський текст, грецькі фрагменти, паралелі, буквальний переклад і коментарі.
3. Оцінити ознаки раннього та пізнього походження.
4. Зібрати зведену таблицю й попередній розподіл шарів.
5. На основі таблиці сформувати власну реконструкцію в `06_Reconstructions/own-model`.
