# Coptic Source Folder

Статус: робочий Unicode-корпус частково зібрано.

У цій папці лежать відкриті робочі матеріали для коптського тексту Євангелія від Фоми. Це ще не фінальний Unicode-корпус коптського тексту.

## Що вже є

- `gospel-thomas-net-home-snapshot.html` - знімок головної сторінки Gospel of Thomas Resources.
- `grondin-interlinear-x_transl-snapshot.html` - індекс інтерлінійного Coptic-English перекладу Michael W. Grondin.
- `grondin-saying-by-saying-snapshot.html` - сторінка входу до saying-by-saying інтерлінійника.
- `grondin-saying-index-snapshot.html` - індекс посилань на 114 логій.
- `grondin-facsimiles-index-snapshot.html` - індекс факсиміле сторінок NHC II, 32-51.
- `grondin_interlinear_logia_001_007/` - перший робочий блок логій 1-7:
  - HTML-сторінки `logion-001-interlinear.html` ... `logion-007-interlinear.html`;
  - відповідні GIF-зображення інтерлінійного тексту `L001.gif` ... `L007.gif`.
- `linssen-thomas-unicode-transcription-snapshot.html` - знімок сторінки Martijn Linssen з повною Unicode-транскрипцією коптського Євангелія від Фоми.
- `linssen-thomas-coptic-unicode-working.txt` - витягнутий робочий текст: пролог `0` і логії `1-114`, один запис на рядок.

## Обмеження

- У Grondin інтерлінійний текст подано переважно як зображення, не як чистий Unicode-коптський текст.
- Linssen прямо заявляє public-domain статус своєї Unicode-транскрипції, але це все одно робоча відкрита транскрипція, не критичне академічне видання.
- Матеріали Grondin придатні для первинного читання й звірки, але не замінюють нормалізований коптський текст.
- Для фінального корпусу бажано отримати нормалізований коптський текст з академічного видання, наприклад Brill Coptic Gnostic Library Online або Layton.

## Наступна дія

Звірити `linssen-thomas-coptic-unicode-working.txt` для логій 1-7 з матеріалами Grondin і, якщо буде доступ, з Brill / Layton.
