# Реєстр джерел

Дата створення: 2026-07-17

Статуси:

- `потрібно знайти` - джерело визначене, але ще не отримане.
- `потрібно перевірити` - джерело знайдене, але не звірене.
- `лише посилання` - повний текст не внесено через права або невизначеність.
- `умовно перевірено` - можна використовувати для попередньої роботи, але бажана звірка.
- `перевірено` - придатне для основної роботи.
