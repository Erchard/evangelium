# План збирання первинного корпусу

Дата створення: 2026-07-17

## Мета етапу

Зібрати не інтерпретації, а робочий корпус текстових свідчень для подальшого аналізу Євангелія від Фоми логія-за-логією. На цьому етапі не робимо реконструкцію, не виставляємо ймовірності логіям і не заповнюємо 114 карток.

## Що потрібно зібрати

### 1. Коптський текст Наг-Хаммаді

- Джерело: Nag Hammadi Codex II, tractate 2.
- Місце в кодексі: NHC II,2, приблизно стор. 32.10-51.28.
- Мова: сахідська коптська.
- Матеріальний рукопис: приблизно IV століття.
- Статус у проєкті: потрібно знайти надійну транскрипцію.
- Бажаний рівень: дипломатична транскрипція + нормалізований коптський текст.
- Основний кандидат: Brill Coptic Gnostic Library Online, якщо буде доступ.
- Обмеження: ймовірно платний або ліцензійний ресурс; не копіювати повністю без перевірки прав.

### 2. Грецькі фрагменти Oxyrhynchus

Потрібні всі три грецькі свідчення:

- P.Oxy. 1
- P.Oxy. 654
- P.Oxy. 655

Попередня робоча інформація:

- P.Oxy. 1: фрагменти логій 26-30, 77, 30-31.
- P.Oxy. 654: логії 1-7.
- P.Oxy. 655: логії 36-40.
- Дата рукописів: приблизно близько 200 року або початок III століття, залежно від фрагмента й джерела.

Статус у проєкті: потрібно перевірити за академічним виданням або надійною базою папірусів.

Бажані джерела:

- первинні публікації Grenfell & Hunt;
- Papyri.info / DCLP, якщо доступний;
- LDAB / Trismegistos для метаданих;
- Gnosis Archive як стартова зручна звірка, але не як єдина основа.

### 3. Контрольні академічні переклади

Потрібні не як основа реконструкції, а як контроль розуміння.

Кандидати:

- Thomas O. Lambdin;
- Bentley Layton;
- Marvin Meyer / Stephen Patterson;
- April DeConick;
- Mark M. Mattison, public-domain translation, як відкрита допоміжна версія;
- K. C. Hanson, як зручна навчальна таблиця з паралелями.

Статус у проєкті: переважно лише посилання або короткі метадані, повні тексти вносити тільки якщо права це дозволяють.

### 4. Робочий український переклад

- Має бути створений окремо після збирання коптського й грецького корпусу.
- Не перекладати з англійського як основного джерела.
- Для кожної логії бажано мати:
  - буквальний переклад;
  - читабельний робочий переклад;
  - коментар до неоднозначностей.

Статус у проєкті: не почато.

### 5. Синоптичні паралелі

Потрібні місця з:

- Марка;
- Матвія;
- Луки.

Рівні роботи:

- список паралелей;
- грецький текст НЗ за обраним виданням;
- український робочий переклад;
- нотатки про можливу залежність, спільну традицію або незалежне формулювання.

Статус у проєкті: потрібно створити таблицю паралелей.

### 6. Інші ранні паралелі

Додавати лише там, де є реальна релевантність:

- реконструйоване Q;
- Дідахе;
- Павло;
- Яків;
- 2 Климента;
- ранні рабинські паралелі;
- Кумран;
- кінічні хреї;
- інші збірки висловів.

Статус у проєкті: другий етап після базового корпусу Фоми й синоптичних паралелей.

## Принципи внесення текстів

- Не вигадувати відсутній текст.
- Не змішувати первинний текст, переклад і реконструкцію в одному файлі.
- Для кожного тексту фіксувати мову, джерело, дату рукопису, дату можливого шару, статус прав і рівень надійності.
- Якщо повний текст не можна легально внести в репозиторій, зберігати тільки посилання, метадані й власні короткі нотатки.
- Для майбутніх карток логій використовувати тільки джерела зі статусом `перевірено` або `умовно перевірено`.

## Мінімальний корпус для першої пробної логії

Для першої пробної картки логії бажано мати:

- коптський текст логії;
- якщо логія входить у P.Oxy. 654, грецький фрагмент;
- 2-3 контрольні переклади;
- синоптичні паралелі, якщо є;
- нотатку про джерело кожного рядка.

Найзручніший стартовий кандидат: логії 1-7, бо для них є P.Oxy. 654.
