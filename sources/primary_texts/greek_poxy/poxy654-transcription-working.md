## Умовні позначки

- `[текст]` - реконструйований / втрачений текст, позначений у TEI як `supplied reason="lost"`.
- `⟨текст⟩` - текст, позначений у TEI як пропущений або доданий.
- `?` - непевне читання (`unclear`).
- `...` - лакуна або місце, яке треба перевірити в XML / факсиміле.
- `вар.` - важливий варіант із критичного апарату DCLP.

Цей файл не замінює XML. Для цитування в аналітичних картках посилатися на XML і, за можливості, звіряти з Grenfell & Hunt / Attridge / Layton.
