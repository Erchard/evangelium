# Arabic Injil Layer Editorial Policy v0.1

Status: project concept accepted, 2026-07-17.

## Purpose

The project may include an Arabic reader layer for the reconstructed sayings gospel under the working title:

> The Reconstructed Injil of Isa ibn Maryam

This layer is intended for comparative, literary, and interreligious presentation. It helps readers who know the Qur'anic and Islamic tradition hear the reconstructed sayings in a high classical Arabic register.

## Non-Negotiable Boundary

This layer must not be presented as:

- a sura of the Qur'an;
- a lost Qur'anic text;
- a revealed scripture;
- the historical Injil claimed by Islam;
- a replacement for the Qur'an, the canonical gospels, or the Coptic Gospel of Thomas.

It is a modern scholarly-literary rendering of the project's reconstructed clean reader.

## Style

Allowed:

- high classical Arabic diction;
- concise, solemn, paratactic phrasing;
- Islamic-friendly naming such as `عيسى ابن مريم`;
- numbered saying units;
- an optional basmala only when clearly framed as a literary/traditional opening, not as a claim of Qur'anic status.

Avoid:

- calling the units `surahs`;
- calling the units `ayahs`;
- using `صدق الله العظيم`;
- adding pseudo-revelatory claims;
- adding material that is not in the reconstructed reader.

Preferred unit label:

- `قول` / `الأقوال`

Avoid as primary labels:

- `سورة`
- `آية`

## File Policy

The Arabic layer should live at:

- `reconstruction/earliest-sayings-gospel/reconstructed-gospel-ar-quranic-register.md`

The file must begin with an editorial note explaining that it is a modern Arabic literary rendering and not Qur'an.

## Translation Policy

The Arabic layer should translate the existing clean reader, not the full Gospel of Thomas and not the entire appendix. It should preserve the project numbering of included logia.

Where the Ukrainian reader has a concise reconstructed core, the Arabic should also remain concise. Do not add interpretive expansions for rhetorical beauty.

## Quality Note

This first Arabic layer is a project draft. Before publication it should be reviewed by someone competent in Classical Arabic and Islamic religious register, especially to avoid accidental Qur'anic mimicry, theological overclaiming, or awkward modern phrasing.
