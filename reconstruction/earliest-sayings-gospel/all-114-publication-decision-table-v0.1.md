| Status | Meaning |
| --- | --- |
| `READER_INCLUDE_MARKED` | Included in clean reader as current marked reconstruction; apparatus must explain uncertainty. |
| `APPENDIX_ONLY_STABLE` | Important or controlled material, but current project model keeps it outside clean reader. |
| `APPENDIX_ONLY_UNCERTAIN` | Interesting but unresolved; appendix explanation required. |
| `DEFER_TO_CLUSTER` | Best handled in thematic/source cluster before publication commentary. |
| `PROMOTE_TO_CONTROLLED_READER_CANDIDATE_PASS` | Not yet in clean reader, but promoted for a later controlled reader pass. |

| `EXCLUDE_AS_SECONDARY` | Likely secondary or outside earliest sayings-gospel reconstruction. |
| `NEEDS_EVIDENCE_BEFORE_FINAL` | Needs direct evidence note, control file, or explicit no-note rationale before publication-level freeze. |
