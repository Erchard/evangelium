## Main Decisions

- Original EUAGELIA reconstruction, translations, commentary, prompts, tables, and methodology remain governed by the commons / anti-ownership policy.
- External sources keep their own rights status.
- Protected modern translations and editions are controls, not base text.
- Paper books must use print-safe references: logion numbers, witness labels, canonical references, short bibliography keys, and printed indexes.
- Digital companion files may preserve repository paths, source snapshots, TEI/XML paths, extraction notes, URLs, and audit trails.
- SBLGNT is a working open canonical Greek control, not a Thomas witness.
- DCLP / Papyri.info XML files are working open papyrus access points, not substitutes for final papyrological checking.
- Hypothetical Greek retroversions are original project scholarship, but not manuscript witnesses.

## Publication Risks Reduced

- The project now has a rights register for major source families.
- The project now has a working bibliography file with short print keys and `VERIFY` markers.
- The project now has a reproducibility note for local source snapshots and temporary extraction risks.
- The paper/digital split is now tied directly to citation and rights policy.
