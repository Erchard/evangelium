# Project

Ця папка містить тільки актуальні керівні документи проєкту.

- `product-spec.md` - специфікація кінцевого продукту.
- `roadmap.md` - дорожня карта.
- `repository-map.md` - розгорнута карта матеріалів.
- `structure.md` - опис структури репозиторію.
- `current-task.md` - поточна черга або активний робочий фокус.

Старі промпти, проміжні звіти й історичні аудити перенесено в `archive/project-history/`.
