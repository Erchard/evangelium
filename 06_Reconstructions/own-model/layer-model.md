# Власна модель шарів

Цей файл призначений для зведення результатів після аналізу окремих логій.

## Шар A: найдавніший можливий фонд

Критерії:

- висока ймовірність походження до 60 року;
- добра відповідність усній формі;
- відсутність потреби в розвиненій церковній або гностичній теології;
- природність у ранньому юдейському або палестинському контексті.

## Шар B: рання редакція / перша збірка

Критерії:

- ймовірність 60-90 років;
- можливе впорядкування незалежних висловів;
- обмежена взаємодія з ранніми християнськими традиціями.

## Шар C: пізніша редакція

Критерії:

- ймовірність 90-130 років;
- можливі сліди Матвія, Луки або іншої літературної взаємодії;
- розвиток езотеричної або аскетичної рамки.

## Шар D: найпізніші нарощення

Критерії:

- ймовірність після 130 року;
- ознаки пізньої редакційної мови;
- слабка відповідність ранньому контексту;
- залежність від уже сформованих богословських дискусій.
