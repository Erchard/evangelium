# Output

Місце для майбутніх експортів і згенерованих публікаційних артефактів.

Робочі тексти поки що живуть у `reconstruction/`; сюди варто складати тільки підготовлені вихідні артефакти, які генеруються або вручну збираються з контрольованого корпусу.

## Структура

- `print/uk/` - майбутня українська паперова книга.
- `print/en/` - майбутня англійська паперова книга.
- `digital/scholarly/` - майбутній цифровий науковий companion із повним evidence apparatus.

Правила для цих виходів визначені в `project/print-and-digital-publication-architecture.md`.
