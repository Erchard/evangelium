# Ukrainian Print Output

Майбутня українська паперова книга.

Ця папка призначена для згенерованих або зібраних print-ready артефактів українського видання. Робочим джерелом істини лишаються файли в `reconstruction/` і `corpus/`.

Паперовий текст має бути self-contained і print-safe: без залежності від клікабельних repository links.
