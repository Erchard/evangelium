# EUAGELIA Commons Dedication and Anti-Ownership Policy v0.1

This project is intended as a contribution to the shared intellectual and spiritual heritage of humanity.

No individual person, organization, publisher, church, academic institution, company, or software system should claim exclusive ownership over the reconstructed text, commentary framework, evidence structure, or scholarly labor created in this repository.

## Permission

Anyone may use, copy, study, quote, adapt, translate, publish, distribute, print, teach from, or commercially use the original project materials created in this repository.

Commercial use is allowed.

No fee, permission request, or institutional approval is required for use of the project's original contributions.

## Required Ethical Condition

Use of this project is allowed only under the same anti-ownership principle:

- do not claim exclusive ownership over the project, its reconstructed text, its commentary method, or its evidence architecture;
- do not assert copyright monopoly over the project materials as if they were the private property of one person or organization;
- do not restrict others from using, copying, translating, publishing, studying, or commercially using the same project materials;
- if you publish a derivative work, make clear that the underlying reconstruction and project framework are offered as common human heritage, not as private intellectual property.

## Attribution

Attribution is appreciated for scholarly transparency, citation, and reproducibility, but attribution is not intended to create ownership, control, or gatekeeping.

Recommended citation:

> EUAGELIA Project, *A Reconstruction of the Earliest Sayings Gospel*, working repository, 2026.

## Third-Party Materials

This policy applies only to original project contributions created in this repository: reconstruction decisions, original translations, commentary, notes, tables, prompts, and project architecture.

It does not override the rights status of third-party materials, including manuscript images, protected academic editions, licensed databases, modern translations, facsimiles, or source texts copied from external providers. Those materials must be used according to their own rights status and citation requirements.

## Publication Note

Before formal print or commercial publication, this policy should be reviewed and, if necessary, converted into a legally precise license by a qualified rights specialist. The project intent is clear: maximum freedom of use, including commercial use, with refusal of exclusive ownership.
